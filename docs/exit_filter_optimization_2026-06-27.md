# 出场与过滤器组合优化记录（2026-06-27）

## 目标

用当前真实纸面交易中的两个入场策略：

- `breakout_retest`
- `volatility_squeeze`

叠加现有两仪四象过滤器，优化出场方式，并通过训练 / 验证 / 压力测试筛选更稳健的组合。

## 数据与方法

- 成本：默认 taker 手续费 `0.06%/边`，滑点 `0.05%/边`
- 风险：沿用当前配置的单笔风险
- 60 天横截面样本：72 个本地缓存山寨币，70% 训练 / 30% 验证
- 180 天样本：12 个本地缓存高波动币，70% 训练 / 30% 验证
- 180 天 walk-forward：切成 3 个时间段逐段验证
- 成本压力：手续费与滑点分别放大到 1.5 倍和 2 倍

## 主要候选出场组合

最终入围组合：

```toml
breakeven_at_r = 0.8
first_take_profit_at_r = 1.5
first_take_profit_fraction = 0.5
trailing_atr_multiple = 2.0
no_progress_bars = 48
minimum_progress_r = 0.15
failed_breakout_confirmation_bars = 1
```

含义：

- 到 `0.8R` 后把止损推到保本附近；
- 到 `1.5R` 先平掉一半；
- 剩余仓位使用较紧的 `2 ATR` 跟踪；
- 48 根 5m K 线内没有达到 `0.15R` 进展则退出；
- 突破失败 1 根确认 K 即退出。

## 验证结果摘要

## 过滤器结论

当前两仪四象过滤器已经接入回测，但本轮证据显示：

- 15m 两仪四象过滤器对交易结果影响很小；
- 4H 两仪四象过滤器在这批样本中没有稳定改善验证集；
- 暂不把 4H 两仪四象过滤器写入实盘配置；
- 后续应单独开发“日线 + 4H + 1H 多周期评分器”，再重新验证。

## 最终判断

入围组合不是在所有样本中收益最高，但它在 60 天大横截面与成本压力下更稳健，同时降低最大回撤。考虑当前系统仍处于纸面测试阶段，优先选择更抗打、更快处理失败突破的出场方式。

因此，本轮落地调整为：

- `breakout_retest` 使用入围出场组合；
- `volatility_squeeze` 使用入围出场组合；
- 两仪四象过滤器保持现有接入方式，不升级为 4H 硬过滤。
